# turkish-poetry-llm-comparison
 Bu proje, Orhan Veli Kanık’ın "İstanbul'u Dinliyorum" şiirine benzer şiirler üreterek, GPT-4 Turbo, Claude, Gemini 2.0 Flash ve Deepseek gibi büyük dil modellerinin Türk edebiyatı bağlamında yaratıcı yazarlık ve edebi çözümleme yeteneklerini karşılaştırmayı amaçlamaktadır.
